							<div id="intro">
								<h3>Version 0.4 DTD</h3>
								<p>The DTD is the format from which all other schemas are derived. It is the only schema that is intended for maintenance.</p>

								<h4 class="subhead">Getting the Files</h4>
								<p>The DTD files are available on the anonymous FTP: <a href="ftp://ftp.ncbi.nih.gov/pub/jats/archiving/0.4/" target="_blank">ftp://ftp.ncbi.nih.gov/pub/jats/archiving/0.4/</a>.</p>
								<p>The schema is also available at the following stable URL: <a href="/archiving/0.4/JATS-archivearticle0.dtd">http://jats.nlm.nih.gov/archiving/0.4/JATS-archivearticle0.dtd</a></p>

								<h4 class="subhead">Schema Structure</h4>
								<p>The main module of the DTD is <span class="code">JATS-archivearticle0.dtd</span>. It imports the subordinate modules. For the schema to work, all modules must be in the same directory.</p>
