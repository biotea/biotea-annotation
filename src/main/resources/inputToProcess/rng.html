							<div id="intro">
								<h3>Version 0.4 RELAX NG</h3>
								<p>The RELAX NG schema has been generated from the Archiving and Interchange DTD for use in RELAX NG tools. <strong>This schema is not intended for maintenance.</strong></p>
								<p>While the structural constraints on document tagging expressed by this schema are identical to those of the DTD, changes to the schema are best made by making the analogous changes to the DTD, and regenerating a schema from the modified DTD.</p>

								<h4 class="subhead">Getting the Files</h4>
								<p>The RELAX NG files are available on the anonymous FTP: <a href="ftp://ftp.ncbi.nih.gov/pub/jats/archiving/0.4/" target="_blank">ftp://ftp.ncbi.nih.gov/pub/jats/archiving/0.4/</a>.</p>
								<p>The schema is also available at the following stable URL: <a href="/archiving/0.4/rng/JATS-archivearticle0.rng">http://jats.nlm.nih.gov/archiving/0.4/rng/JATS-archivearticle0.rng</a></p>

								<h4 class="subhead">Schema Structure</h4>
								<p>The main module of the RELAX NG schema is <span class="code">JATS-archivearticle0.rng</span>. It imports the subordinate modules. For the schema to work, all modules must be in the same directory.</p>
